Esta aplicação web é o resultado do segundo projeto realizado no âmbito da unidade curricular Engenharia de Dados e Conhecimento. O tema, assim como no primeiro projeto, baseia-se na exposição de informação de dados geográficos sobre o país, juntamente com algumas das suas atrações turísticas.


Ferramentas utilizadas:

  • Python/Django para programação da aplicação.
  • RDF(utilizado formato NT).
  • Triplestore GraphDB para repositório de dados.
  • SPARQL para pesquisa e alteração de dados na triplestore.
  • SPARQLwrapper para integração de dados da Wikidata, para complementar a informação do nosso repositório.
